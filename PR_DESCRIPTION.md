# הערות טליה — שיפורים למערכת

## מה השתנה

### 🔧 סטטוסים מפושטים
- הורדת סטטוס "נפתרה" — עכשיו יש רק **פתוחה** / **סגורה**
- מיגרציה אוטומטית של ה-DB: כל "Resolved" הקיימים הועברו ל-"Closed", "InProgress" ל-"Open"

### 🗄️ טבלת `lockers` חדשה
מאגר לוקרים מלא עם:
- `locker_id`, `school_name`, `cabinet_name`, `cell_number`
- `lock_type` (digital/mechanical), `student_code`, `master_code`, `lock_number`
- `current_student_id`, `status` (available/in_use/faulty)
- `academic_year`, `assigned_at`, `notes`

Seed של **270 לוקרים דמה** (קובץ `seed_lockers.py`):
- 13 בתי ספר — 6 דיגיטליים, 5 מכניים, 2 מעורבים
- ארונות לפי שמות פרחים (נרקיס 1, כלנית 1...)
- תאים בתבנית 101-303 (9 תאים לארון)
- קודים דיגיטליים: 6 ספרות | קודים מכניים: 3 מספרים 00-39 בפורמט `XX-XX-XX`
- מספרי מנעול: לרוב 8 ספרות

### 🎯 חיבור אוטומטי לוקר↔תלמיד בדיווח תקלה
כשבוחרים תלמיד, המערכת מושכת אוטומטית את פרטי הלוקר שלו:
- מספר לוקר, ארון, תא, סוג נעילה, קוד תלמיד, קוד מאסטר / מספר מנעול
- שדות "מספר לוקר" ו"סוג הלוקר" מוסתרים לגמרי (אין צורך למלא ידנית)
- רשימת סוגי התקלה נטענת אוטומטית לפי סוג הנעילה
- תלמיד ללא לוקר → חוזרים למילוי ידני עם אזהרה

### 📊 מלבני סטטיסטיקה לחיצים
- לחיצה על "פתוחות" / "ספרים נעולים" / "סגורות" / "סה"כ" מסננת את הטבלה
- המלבן הפעיל מודגש בכחול
- הספירות תמיד על כל הנתונים, לא קורסות כשמסננים

### 🔀 מיון חדש (dropdown)
8 אפשרויות: תאריך (חדש/ישן), שם תלמיד, בית ספר, חומרה, דחיפות, תקלות חוזרות, סוג תקלה

### 🔒 פרטי הלוקר במודל התקלה
כשפותחים תקלה קיימת — מופיע בלוק "פרטי הלוקר" עם ארון + תא + קודים, כדי שהטכנאי ידע לאן ללכת.
עבור תקלות ישנות בלי `locker_id`: fallback ללוקר הנוכחי של התלמיד.

### 🕐 תיקון timezone
תאריכים נשמרו ב-UTC אבל הדפדפן קרא אותם כ-Local → פער של 3 שעות. תוקן ע"י סימון `Z` ב-JSON.

### 📞 טלפון הורה
נוסף טלפון הורה לכרטיס מידע התלמיד וגם לפרטי התקלה.

## איך למשוך ולבדוק

```bash
git fetch origin
git checkout shilo-improvements
pip install -r requirements.txt
python flask_app.py
```

ואז לפתוח http://localhost:5000

> הערה: הקובץ `faults_system.db` כולל את הטבלה החדשה + נתוני דמה. אם תרצה לשחק מאפס:
> ```bash
> python seed_lockers.py
> ```
